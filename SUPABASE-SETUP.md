# FlowSpecta - Setup Supabase Próprio

## 1. Criar Conta Supabase
- Ir em: https://supabase.com
- Criar conta (gratuita)
- Criar novo projeto: "flowspecta-production"

## 2. Configurar Database
- Copiar schema de: supabase-schema-completo.sql
- Executar no SQL Editor do Supabase

## 3. Configurar Edge Functions
- Instalar Supabase CLI
- Deploy das 3 funções principais

## 4. Obter Credenciais
- Project URL
- Anon Key
- Service Role Key

## 5. Configurar .env Local
VITE_SUPABASE_URL=[sua URL]
VITE_SUPABASE_ANON_KEY=[sua chave]
