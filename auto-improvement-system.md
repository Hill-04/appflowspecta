# SISTEMA DE AUTO-APRIMORAMENTO - FLOWSPECTA

## CICLO DE APRENDIZADO

| Etapa | Quando | Ação |
|---|---|---|
| Coleta | Semana 1-2 | Prospectar normalmente, usar todos os 4 scripts igualmente |
| Análise | Semana 3-4 | Identificar script com melhor resposta, nicho mais receptivo |
| Otimização | Mês 2 | 60% script vencedor, 40% outros; 70% nicho vencedor |
| Refinamento | Mês 3+ | A/B testing contínuo, documentar cases de sucesso |

---

## MÉTRICAS RASTREADAS

### Performance de Scripts
- DMs enviadas por script (S1/S2/S3/S4)
- Taxa de resposta por script
- Taxa de conversão por script
- **Ação:** Usar mais o script vencedor, testar variações do pior

### Performance de Nichos
- Taxa de resposta por nicho (Dentistas/Advogados/Fitness)
- Ticket médio por nicho
- Tempo médio de fechamento
- **Ação:** Priorizar nicho com melhor ROI

### Timing Ótimo
- Horário de envio vs taxa de resposta
- Dia da semana vs engajamento
- Tempo ideal para follow-up
- **Ação:** Concentrar envios nos horários de pico

### Funil de Conversão
- Taxa entre cada etapa do Kanban
- Etapa onde mais leads travam
- Tempo médio em cada etapa
- **Ação:** Criar scripts específicos para gargalos

## RELATÓRIOS

### Diário (5 min)
- Quantas DMs enviadas?
- Quantas respostas?
- Próxima ação prioritária?

### Semanal (15 min)
- Meta 75 DMs atingida?
- Script com mais resposta essa semana?
- Orçamentos enviados?

### Mensal (30 min)
- Script e nicho vencedor do mês
- Ajustes para o próximo mês
- Projeção de receita
